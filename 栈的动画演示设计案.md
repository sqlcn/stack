## 类的设计
```
# 桶
    class Tube:
        属性：
            一个模拟桶容器的数组
        方法：
            进栈
            出栈
            画出自己（和桶里的球）
# 球
    class Ball:
        属性：
            球代表的数
            球在屏幕上的坐标
        方法：
            画出自己
```
